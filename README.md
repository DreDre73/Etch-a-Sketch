# Etch-a-Sketch
Etch-a-Sketch web app with adjustable grid size and click-drag drawing.

## Features
- Dynamic grid creation with user-defined size.
- Draw by hovering or pressing-and-holding.
- Reset the grid at any time.

## Technologies
- HTML
- CSS
- JavaScript
